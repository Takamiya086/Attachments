## Overview

By default, only positive prompts are included; negative prompts will be listed separately.

If you don't want to use the web interface, you can call the API provided by ComfyUI.

---

## :warning: rules for censoring generated images:

:white_check_mark: **censored**: Suitable for all ages, no violent, bloody, or sexual content.

:interrobang: **questionable**: Sexually suggestive, but without directly exposing genitals.

:x: **explicit**: Explicit images with exposed genitals.
